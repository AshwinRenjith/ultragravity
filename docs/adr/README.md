# Architecture Decision Records (ADR)

This directory stores architecture decisions for Ultragravity.

## Format
Each ADR should include:
- Status
- Context
- Decision
- Consequences
- Alternatives considered

## Naming
Use zero-padded numbering:
- `0001-...md`
- `0002-...md`

## Lifecycle
- Start with `Accepted` when merged.
- Add superseding ADR when replacing an earlier decision.
